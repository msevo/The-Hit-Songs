About thehitsongs.com

TheHitSongs.com uses the KNN (K-Nearest Neighbors) machine learning algorithm to analyze your mp3 file and classify it as a hit song or not a hit song. We uploaded songs from the top 100 lists of the past twenty years and analyzed their bpms (beats per minute), average frequencies, mode frequencies, and durations. Then we fed the data into the machine learning algorithm, letting it know that these songs should be considered "hits". We proceeded to do the same with songs that were not considered "hits" during their respective years. We fed these songs into the machine learning algorithm, and made sure it knew that these songs should not be considered "hits". This is the process by which we trained the algorithm.

We built this program with the purpose of illuminating the music-industry's rubric for what is considered to be "good" music. If the program generated the "not a hit" response for your song, it only means that the data representing your song does not fit into the schemas of the most popular songs.
